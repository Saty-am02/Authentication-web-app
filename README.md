# Authify - Full Stack Authentication Platform

A secure authentication system developed using **Spring Boot 3** and **React**.

## Tech Stack
* **Backend:** Java 17, Spring Boot, Spring Security, JWT, JPA/Hibernate, MySQL
* **Frontend:** React.js, Tailwind CSS, Vite
* **Email Service:** Brevo (formerly Sendinblue) with Thymeleaf templates

## Features
* JWT-based authentication
* Email OTP verification
* Password reset functionality
* Welcome email automation
